# ToDo App with Projects & Tags

プロジェクトごとにタスクとタグを管理できるToDoアプリ。

## 画面構成

```
┌─────────────────────────────────────────────────────────────┐
│                        App                                   │
├─────────────────────────────────────────────────────────────┤
│  [ダッシュボード] [タスク一覧] [タグ管理]    プロジェクト: ▼  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                      各画面の内容                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 画面遷移

```
                    ┌─────────────┐
                    │  Dashboard  │
                    │     (/)     │
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              │            │            │
              ▼            ▼            ▼
       ┌──────────┐ ┌──────────┐ ┌──────────┐
       │ TodoList │ │ TagList  │ │ (Nav)    │
       │ (/todos) │ │ (/tags)  │ │          │
       └────┬─────┘ └────┬─────┘ └──────────┘
            │            │
            ▼            ▼
     ┌────────────┐ ┌────────────┐
     │ TodoDetail │ │ TagDetail  │
     │(/todos/:id)│ │(/tags/:id) │
     └────────────┘ └────────────┘
```

## 機能

- **プロジェクト管理**: 複数プロジェクトを切り替えて管理
- **タスク管理**: 追加・完了・削除・編集
- **タグシステム**: ラベル・担当者・優先度の3カテゴリ
- **フィルタリング**: ステータス・タグでタスクを絞り込み
- **永続化**: ブラウザを閉じてもデータが残る
