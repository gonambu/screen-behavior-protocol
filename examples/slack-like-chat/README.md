# Slack-like Chat

Slackライクなチャットアプリケーション。リアルタイムメッセージング、チャンネル管理、スレッド機能を含む。

### ワークスペースバー

```
┌────┐
│ A  │  ← ワークスペースA（選択中）
├────┤
│ B  │  ← ワークスペースB
├────┤
│ C  │  ← ワークスペースC（未読あり：●）
├────┤
│ +  │  ← ワークスペース追加
└────┘
```

## 機能

- **ワークスペース**: 複数ワークスペースの切り替え
- **チャンネル**: パブリック/プライベートチャンネル、作成・参加
- **DM**: ユーザー間のダイレクトメッセージ
- **メッセージ**: 送信・編集・削除、メンション、添付ファイル
- **スレッド**: メッセージへの返信をスレッド表示
- **リアクション**: 絵文字リアクション
- **検索**: メッセージ・チャンネル・ユーザーの検索
- **リアルタイム**: WebSocketによるリアルタイム更新
- **タイピング表示**: 入力中のユーザー表示
- **未読管理**: チャンネル・DMごとの未読数表示
